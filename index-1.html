Programme

Structure
3× full body + 1× CrossFit WOD per week
Session layout
10 min cardio · 2 chest · 2 back · 3 legs · 2 abs · cooldown
Restrictions
No overhead press · No wide-grip pulling · No pull-ups · No hanging core work
Equipment
Dumbbells, kettlebells, bands, rings — machines listed only as backups

Overload rules

1. All sets clean → add 2.5 kg upper / 5 kg lower next session.
2. Can't add weight → add 1 rep per set first.
3. Stuck 2 weeks → slow eccentric to 3s.
4. Reps/weight both stuck → add a set (5 is the ceiling).
5. Every 4th week = deload. Drop loads 40%.

Rest times

Legs / main compounds: 90–120 s
Chest & back supersets: 60–75 s
Abs: 45–60 s
